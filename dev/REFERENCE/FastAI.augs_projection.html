FastAI.augs_projection

augs_projection([; kwargs...])

Helper to create a set of projective transformations for image, mask and keypoint data. Similar to fastaiâ€™s aug_transforms.

Keyword arguments

  • flipx = true: Whether to perform a horizontal flip with probability 1/2. See FlipX.

  • flipy = false: Whether to perform a vertical flip with probability 1/2. See FlipY.

  • max_zoom = 1.5: Maximum factor by which to zoom. Set to 1. to disable. See FastAI.Zoom.

  • max_rotate = 10: Maximum absolute degree by which to rotate. Set to 0. to disable. See FastAI.Rotate.

  • max_warp = 0.05: Intensity of corner warp. Set to 0. to disable. See WarpAffine.
