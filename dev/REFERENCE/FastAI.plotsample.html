FastAI.plotsample

plotsample(method, sample)

Plot a sample of a method.

Learning methods should implement FastAI.plotsample! to make this work.

Examples

using FastAI, Colors

sample = (rand(Gray, 28, 28), 1)
method = ImageClassification(1:10, (16, 16))
plotsample(method, sample)
