StatsBase.nobs

nobs(model::StatisticalModel)

Return the number of independent observations on which the model was fitted. Be carefulwhen using this information, as the definition of an independent observation may varydepending on the model, on the format used to pass the data, on the sampling plan(if specified), etc.
